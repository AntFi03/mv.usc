# mv.usc

<!-- badges: start -->

<!-- badges: end -->

The goal of mv.usc is to serve as a set of shortcut functions that
perform elementary multivariate statistical tests. It intended to be
used as a tool in M.Sc. courses by allowing for checks over the tests
results.

## Installation

You can install the development version of mv.usc from GitHub with:

``` r
pak::pak("antfi03/mv.usc")
```

If you want to install an specific version, you can do so with:

``` r
pak::pak("antfi03/mv.usc@v0.0.1")
```
